# Hospital Consultation System

ASP.NET MVC web application designed to manage hospital consultations and patient appointments.

The system supports the management of hospitals, specialties, schedules, payments, and appointment confirmations, focusing on structured backend logic and data modeling.

## Technologies
- C#
- ASP.NET MVC (.NET Framework)
- Entity Framework
- SQL Server
- HTML, CSS, JavaScript

## Architecture
- MVC (Model-View-Controller)
- Server-side rendering with Razor Views
- Database access using Entity Framework

## Project Type
Academic / Practice project

## Focus
- Backend development
- Data modeling
- MVC architecture
- Practical software implementation
